# 🚗 Traffic AI Detection with Speed Estimation

An intelligent traffic monitoring system that detects vehicles and estimates their speed using computer vision and deep learning.

## Features

- **Vehicle Detection**: Uses YOLO and CNN models for accurate vehicle detection
- **Speed Estimation**: Estimates vehicle speed using optical flow and object tracking
- **Real-time Processing**: Process video streams in real-time with configurable frame skipping
- **Interactive Dashboard**: Streamlit-based interface with live video feed and analytics
- **Multiple Input Sources**: Support for webcam, uploaded videos, and sample videos
- **Export Reports**: Generate and download traffic analysis reports
